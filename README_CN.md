# Hexo Fust 主题

超级简约的主题, 基于 even.

## 预览

具体样式可以访问我的博客： [https://blog.rekols.com](https://blog.rekols.com/)

## 特色

* Ubuntu 字体

* 高度定制

## 安装

```
$ npm install hexo-renderer-scss --save
$ git clone https://github.com/rekols/hexo-theme-green.git themes/green
```

## 更新

```
$ cd themes/green
$ git pull
```
